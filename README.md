Play piano by moving your fingers in the air! A gesture-controlled virtual instrument built with Computer Vision.

https://github.com/keerthi13k8/Touchless-music

## ✨ Features

- **Real-time hand tracking** - 21 landmark points with <50ms latency
- **4 Musical Instruments** - Piano, Guitar, Flute, Synth
- **Visual Effects** - Particle bursts, fingertip trail, reactive background
- **Guided Song Mode** - Learn "Twinkle Twinkle" and "Happy Birthday"
- **Color-changing keys** - Green when pressed, Yellow for next note in song mode
